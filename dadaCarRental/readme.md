## 嗒嗒租车系统
运用了Java**面向对象**的知识：

* 抽象类
* 封装
* 继承
* 多态

完成一个简易的租车小程序

## 业务需求
1. 展示可租用车型
2. 用户选择租用车辆
3. 计算总价并输出账单信息
